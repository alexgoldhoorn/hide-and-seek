Hide-and-Seek Robot Search Simulator

C++ simulation framework for probabilistic robot search and tracking with multiple solver implementations. Includes POMCP, MOMDP, Particle Filters, Kalman Filters, and heuristic methods. This is the complete research code from my PhD thesis on searching and tracking humans in urban environments.

Quick Start

The simulator can be built as a library or with Qt for the graphical interface:

# Build library version
cd trunk/build
cmake ..
make

# Or build Qt version for GUI/multiplayer
cd trunk
qmake hsmomdp.pro && make    # Automated seeker
qmake hsserver.pro && make   # Game server
qmake hsclient.pro && make   # GUI client

Note: This code is from 2011-2016 research and uses historical dependencies (g++ 4.8, OpenCV 2.3, Qt 5, MySQL 5.6). See the README for modern build guidance.

Example Usage

Run a POMCP-based seeker in hide-and-seek mode:

# Start game server
./bin/hsserver

# In another terminal, run POMCP seeker on map 1
./bin/hsmomdp -Gh -scd -A 1 -u TestUser

# Flags:
#   -Gh   : Hide-and-seek game mode
#   -scd  : POMCP solver with discrete states
#   -A 1  : Use map ID 1
#   -u    : Username for logging

The simulator supports many solvers and configurations. See trunk/README.txt for the complete command-line reference (17KB documentation).
